📌 3월 21일 수업 중 과제 업로드: 


- cat: 파일 내용 확인
- cat > 파일명: 파일 내용 수정 // 수정 후 저장하고 나가려면 Ctrl+D
- head 파일명: 파일 내용의 첫 10줄 출력
- tail 파일명: 파일 내용의 마지막 10줄 출력

📌 3월 28일 수업 중 과제 업로드: 

- mv : 파일 및 디렉토리 이동/이름 변경
- mv 파일명 새파일명: 같은 디렉토리 내에서는 파일명을 변경
- ln -s 파일명 바로가기파일명: 바로 가기 파일 생성. 즉 원본 파일을 변경하면 바로가기 파일도 같이 변경
- cp 파일명 새파일명: 파일을 새 파일로 복사
